ID3v1.0 Tag

An ID3v1.0 tag is 128 bytes of text divided into 6 fields which can be added on the end of any MPEG audio file (*.mp1, *.mp2, or *.mp3). In theory, unused bytes should be set to 0x20 (ASCII 'space') not 0x00 (binary zero).

Bytes (0 - 2)

3 - 32

33 - 62

63 - 92

93 - 96

97 - 127

'TAG'

Track Name

Artist Name

Album Name

Year

Comment
